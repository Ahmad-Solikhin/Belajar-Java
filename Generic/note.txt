Menambahkan parameter typr di parameter
Dengan generic bisa diubah tipe datanya sesuai dengan yang ditentukan oleh kita

Pake generic nanti tipe datanya bisa dicek saat kompilasi

Generic itu sama aja kayak general yang bisa digunakan oleh tipe data apapun

Menggunakan parameter type dengan cara <tipe_data>. bisanya ini di function maupun class digunakan T
tipe data yang dimaksudkan harus objek

Generic class merupakan class atau interface yang memiliki parameter type
Generic parameter type yang biasa digunakan sebagai berikut:
- E = element atau collection
- K = key
- T = Type
- V = Value

Ini bisa dilakukan lebih dari satu key, namun menggunakan nama key yang berbeda dari yang sebelumnya

Generic ini juga bisa dipake di method atau function

=================================================
Invariant

Dalam invariant gada konsep inheritance, misal membuat object dengan generic data Object da mengisinya dengan String, maka tidak bisa

=================================================
Covariant

Untuk melakukan polymorphism di generics bisa menggunakan covariant ini
Caranya dengan menggunakan (? extends ParentClass), sehingga kita bisa memasukkan object turunan dari parentclass nya
Tapi ini hanya read only, di mana tidak bisa dilakukan manipulasi datanya di dalamnya
Hanya read-only karena pada memanipulasi maka akan terkena invariant

=================================================
Contravariant

Ini sama kayak covariant tapi dibalik, di mana dari parent ke child
Caranya (? super childClass), contravariant bisa melakukan write dan read, namun hati2x dalam read

=================================================
Bounded Type Parameter

Untuk membatasi type yang diperbolehkan dalam generics dapat menggunakan fitur Bounded Type
Sebagai contoh jika ingin number bisa menggunakan <T extends Number>
Jika ingin menambahkan lebih dari 1 bisa dengan menambahkan &, namun cuma 1 yang boleh class, sisanya adalah interface

=================================================
Wildcard

Ini bisa digunakan jika tidak peduli dengan tipe data pada generics nya dan hanya ingin memprint nya saja bisa menggunakan <?>
Tipe data dari wildcard ini adalah Object

=================================================
Type Erasure

Proses pengecekan generics pada compile, dan dihiraukan pada runtime
Nantinya pada binary file generics akan diubah menjadi Object

=================================================
Comparable dan Comparator

Comparable bisa di implements pada class yang dibuat untuk membandingkan object
Jika class nya adalah milik orang lain dan tidak bisa dilakukan perubahan bisa menggunakan Comparator
