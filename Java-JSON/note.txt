----------------------------------------------
Deserialization
Jakson memiliki banyak fitur yang bisa diaktifkan atau di aktifkan
Deserialization ini adalah proses membaca JSON menjadi Object Java

----------------------------------------------
Serialization
Ini dari Object ke JSON

----------------------------------------------
Serialization Inclusion
Secara default jika isi object nya null maka akan tetap dibuat Json
dengan incluseion bisa menentukan mana yang wajib ada dan tidak harus dimasukkan jika null

----------------------------------------------
Date Time
JSON tidak memiliki tipe data waktu
Biasanya untuk tipe data date time menggunakan milis atau menggunakan string
Secara default jackson menampilkan dalam format milis
Jika ingin menggunakan string, harus dilakukan konfigurasi pada object mapper nya. Harus diberitahu juga date formatnya dan menonaktifkan fitur milis

----------------------------------------------\
Jackson Annotation
Ada beberapa annotation yang bisa mengubah behaviour nya
- @JsonIgnore, menandai field yang akan di ignore
- @JsonFormat, untuk mengubah misal date time
- @JsonProperty, untuk mengubah attribute ketika proses serialization dan deserialization
